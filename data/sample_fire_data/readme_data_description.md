Sample tensors illustrating the data format used in the STGL-SAFE framework:

- fire_intensity.npy: shape (T, H, W), approximate heat release / intensity field.
- smoke_concentration.npy: shape (T, H, W), smoke concentration field.
- wind_field.npy: shape (T, H, W, 2), horizontal wind vectors (u, v).
- terrain_dem.npy: shape (H, W), digital elevation model.

All arrays are small random examples for demonstration only.
